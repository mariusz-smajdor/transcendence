# Transcendence

A full-stack real-time web application built as part of the 42 curriculum.  
The project is a multiplayer Pong platform featuring authentication, matchmaking, chat, and game rooms with real-time gameplay.

---

## 🚀 Overview

**Transcendence** is a web-based multiplayer gaming platform where users can:

- Create accounts and authenticate securely
- Play real-time Pong matches against other players
- Join matchmaking queues or private rooms
- Chat with other users (public & private messaging)
- Track statistics, rankings, and match history
- Customize user profiles

The goal of this project is to explore full-stack development, real-time communication, and scalable architecture.
